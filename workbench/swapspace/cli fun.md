# Motivate

```
$ git clone https://github.com/mubaris/motivate.git
$ cd motivate/motivate
$ sudo ./install.sh
$ source ~/.bashrc
```

zsh users should replace `.bashrc` with `.zshrc`.

### Update Database

```
$ git clone https://github.com/mubaris/motivate.git
$ cd motivate
$ ./UPDATE
```

### Usage

```
$ motivate

"When something is important enough, you do it even if the odds are not in your favor."
```
